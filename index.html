My projects

Here are things which I want to show you the most. More to explore is available here.

Daily quotes

My first project with Javascript. It's based on two arrays - quotations and authors. Quote is created by Math.random() method. The assumption was to generate one quote a day, so the generation button disappears after the quote is drawn. Also added some visual effects and pop-up with copyright info. App is in Polish language.

Interior Designer Website

It meant to be "visiting card" website for interior designer. It uses fullpage.js library for full page concept. Also it allows to create great transitions on scrolling so I wanted to utilize it.

Today's Weather

My first bigger project using Javascript. App which allows you to check the weather of all countries in the world. It's basically made with udemy course, but I made new design and added a lot of new stuff. More detailed information about it you can find in README file on GitHub.

Online Webinar Page

Really simple project which arose becuse I wanted to learn Figma. I created layout by means of this design tool and wanted to encode it. It contains simple transitions on page and custom form.
